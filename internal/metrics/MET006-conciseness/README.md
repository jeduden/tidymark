# MET006: conciseness

Heuristic conciseness score (`0` to `100`, lower is less concise).

- **Scope**: file
- **Sort default**: ascending
- **Type**: float (1 decimal place)

## Notes

Combines lexical density, filler usage, sentence length,
and verbose phrase penalties.
